## run ollama server

ollama serve

## list all downloaded models

ollama list

## download models (example)

ollama pull qwen2.5-coder:14b

## delete all models (one-liner)

ollama list | awk 'NR>1 {print $1}' | xargs -r -n1 ollama rm

## communicate with models (example)

ollama run qwen2.5-coder:14b
